# CIS - Phase 1: User Management API

REST API for user management (CRUD) that coexists with the legacy CLI system.

Technologies:
- Java 21 (LTS)
- Spring Boot 3.4.x
- Maven
- Spring Data JPA + Hibernate
- MySQL (shared table)
- JUnit 5 + Mockito + AssertJ

## Quick Setup
1. `mvn clean install`
2. `mvn spring-boot:run`

See the wiki for more details.
